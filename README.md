# swift-learning

My journey learning **Swift, SwiftUI and Xcode** — one concept at a time. Each concept gets its own folder containing the lesson code plus a small validation project to lock it in.

> Mon apprentissage de Swift, SwiftUI et Xcode, une notion à la fois : un dossier par notion, avec le code des leçons et un projet de validation.

## How it's organised

One folder per concept (`NotionXX-...`), in order:

```
swift-learning/
├── Checkpoint01-Integration-N1-N5/
├── Notion01-Bases-Syntaxe/              # basics & syntax
├── Notion02-Strings-Nombres-Booleens/   # strings, numbers, booleans
├── Notion03-Conditions-boucles/         # conditionals & loops
├── Notion04-Fonctions-Paramètres/       # conditionals & loops
└── Notion05-Collections/        # conditionals & loops
```

Each folder holds the lesson code and a short practice project that applies it.

## Goals

- Build solid Swift fundamentals before moving into full SwiftUI apps
- Keep a clear, reviewable trail of progress, concept by concept
- Practise in Xcode with small, self-contained exercises

## Tech

- **Swift**
- **SwiftUI**
- **Xcode**
